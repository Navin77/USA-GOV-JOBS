# American Gov Jobs
A react app to search current job openings in American goverment. It uses the open api provided by search.gov

This project was made with react and redux.

Live Demo: https://navin77.github.io/USA-GOV-JOBS/

## Includes
* React
* Redux
* Redux-form
* enzyme
